jquery-ui-touch-punch is designed to enable jQuery UI functionality for touch devices. http://touchpunch.furf.com/

Chrome Mobile for Android breaks the functionality of the .click() method for elements with the .draggable() method attached to them. This includes elements nested within the draggable div.

Chrome appears to be more sensitive than the other browsers to the 'touchmove' event and is causing touch-punch's simulated click event to be cancelled.

I have recreated this on a Nexus 9 running Android 5.0.1 & Samsung SM-T520 running 4.4.2
